# Evidencias del modelo

Esta carpeta contiene evidencia visual del funcionamiento del modelo de detección de EPP entrenado con YOLOv8.

Aquí se incluyen:

- Ejemplos de anotaciones del dataset.
- Predicciones del modelo sobre imágenes del conjunto de validación.
- Predicciones del modelo sobre imágenes nuevas no utilizadas en el entrenamiento.

Estas evidencias permiten evaluar visualmente el desempeño del modelo y complementar las métricas obtenidas durante la validación.
