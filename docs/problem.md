# Problema

El sector AECO presenta una alta incidencia de accidentes laborales en obra. Muchos de estos accidentes se relacionan con la falta de uso de Equipos de Protección Personal (EPP) y con la escasa trazabilidad de condiciones de seguridad.

Este proyecto explora el uso de un modelo YOLOv8 para detectar personas y elementos de seguridad en imágenes de obra.
