# Checklist de gobernanza

## Privacidad y consentimiento
- Origen del dataset documentado
- Uso académico del dataset aclarado

## Limitaciones
- El modelo no debe usarse como sistema autónomo de seguridad
- Requiere supervisión humana

## Riesgos
- Los falsos negativos pueden omitir casos reales de ausencia de EPP
- Los falsos positivos pueden inducir interpretaciones incorrectas
