## Derechos del dataset

El dataset utilizado en este proyecto fue construido manualmente a partir de imágenes públicas obtenidas desde la plataforma Unsplash.

Fuente de imágenes:
https://unsplash.com/s/photos/construction-person-equipment

Las imágenes fueron seleccionadas manualmente y posteriormente anotadas utilizando Roboflow para crear un dataset de detección de objetos en formato YOLOv8.

Tipo de dataset:
- Dataset construido a partir de imágenes públicas

Licencia de las imágenes:
Las fotografías provienen de Unsplash y están distribuidas bajo la Unsplash License, que permite su uso libre, incluyendo modificación y uso en proyectos académicos.

Las anotaciones, estructura del dataset y entrenamiento del modelo corresponden a trabajo propio realizado para este proyecto académico.
